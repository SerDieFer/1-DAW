Bienvenidos al módulo de lenguaje de marcas

Los contenidos de este ejercicio tratan de ilustrar el funcionamiento de las etiquetas de formato del texto con HTML. Para
ello, se utiliza como base el documento HTML del ejercicio anterior.

Encabezados

Los encabezados tienen como función estructurar el contenido del documento en partes para
identificar los apartados del texto. Normalmente, se utiliza un encabezado de nivel 1, que corresponde con el de mayor
tamaño, para identificar los apartados principales. Se pueden ir utilizando otros encabezados de menor tamaño para indicar los
subapartados. Existen 6 tipos de encabezados.
